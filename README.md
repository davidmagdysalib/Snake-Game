<h1>🐍 Snake Game</h1>

A classic Snake Game built from scratch using Python and the Turtle graphics module.

<h2>Features</h2>

- Smooth snake movement
- Food spawning system
- Score tracking
- Snake growth mechanics
- Wall collision detection
- Self-collision detection
- Clean object-oriented design

<h2>Technologies</h2>

- Python 3
- Turtle Graphics

<h2>How to Run</h2>

1. Clone the repository.
2. Run the main Python file:
   python main.py

<h2>Preview</h2>

Control the snake, collect food, grow longer, and try to achieve the highest score without crashing into the walls or yourself.
